Notes:

  • Only .xlsx format Excel files are accepted.
  • adi (product name) and brend_kod columns must be filled. If both are empty or "nan", that row will not be accepted.
  • If columns like firma, kateqoriya, avtomobil, vitrin exist, they will be created automatically or the existing one will be selected.
  • maya_qiymet and qiymet columns must be numeric (can be written with dot or comma).
  • stok must be an integer.
  • All columns shown in the sample must exist in your Excel file and have the same names (order does not matter).
  • If a product with the same brend_kod, firma, and sahib exists, that product will be updated. If brend_kod is the same but firma is different, a new product will be created. So, you can add multiple products with the same brend_kod and different brands.
  • When importing from Excel, the brand of an existing product is not changed. Only other information is updated.
  • Error messages will be shown for empty or incorrectly filled rows.
  • Only verified users can import products.
